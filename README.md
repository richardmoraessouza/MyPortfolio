## 🚀 Como Executar

```bash
# Clone o repositório
git clone https://github.com/richardmoraessouza/Portf-lio.git
cd Portf-lio/portfolio

# Instale as dependências
npm install

# Execute em desenvolvimento
npm run dev

# Acesse http://localhost:5173
```
